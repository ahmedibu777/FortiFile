Get Started

Option A: Download & Run (Easiest)

  1. Go to Releases page
  2. Download FortiFile-1.0.0.jar
  3. You're done! Run it: java -jar FortiFile-1.0.0.jar

Console Mode

java -jar FortiFile-1.0.0.jar

This opens the interactive menu where you can encrypt/decrypt files and see live progress.

GUI Mode

java -jar FortiFile-1.0.0.jar --gui

Opens the Swing GUI for point-and-click encryption with file dialogs.

Build from Source

git clone https://github.com/ahmedibu777/FortiFile.git cd FortiFile mvn clean package

Requires Java 21 JDK and Maven 3.9+

FAQ

Q: Can I use this on macOS or Linux?
A: Yes! FortiFile runs on any operating system with Java 21 installed.
Q: What if I forget my password?
A: Unfortunately, there's no "forgot password" option. Choose a password you'll remember!
Q: How long does encryption take?
A: For most files up to 1 GB, a few seconds. For larger files, it depends on your disk speed. The progress bar keeps you updated.
Q: Is my data sent to the cloud?
A: No. Everything happens locally on your computer. Your files never leave your machine.
